# act-for-react
这是一个react的脚手架，用于快速创建react应用。

## 特点
1. 拥抱新技术，体验最新稳定版本的React，Webpack,React-router等技术。

当前版本：
React    v16.0.0
Webpack  v3.7.0
React-router v4.2.0
ES6（通过babel的env配置 支持最新版es6，如:es2017）

## 使用到的技术：

https://github.com/SBoudrias/Inquirer.js
